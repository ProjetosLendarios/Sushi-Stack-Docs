# SushiStacker-Docs

Website promocional e páginas legais do jogo `Sushi Stack`.

## Conteúdo

- `index.html` - landing page principal do jogo
- `creditos.html` - página de créditos do projeto
- `privacidade.html` - política de privacidade para publicação e Play Store
- `styles.css` - estilos globais do site
- `script.js` - interações leves do site

## Identidade visual usada

O site usa assets reais do projeto:

- `Res/Fonts`
- `Res/Icon`
- `Res/PlayStore`
- `Res/img/Prints`
- `Res/GoncaloGarrido`
- `Res/Sons/V1`

## Estado atual

O website está preparado para:

- apresentar o jogo `Sushi Stack`
- mostrar screenshots reais
- destacar o autor `Gonçalo Garrido`
- publicar créditos e política de privacidade
- servir de base pública para a ficha da Google Play
